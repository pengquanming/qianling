挂一家辣鸡公司“兴唐IT教育”

发表于 2019-12-04 更新于 2019-12-06 分类于 展示 阅读次数 评论数

　　第一次参加商业组织的比赛，就踩雷了……
　　说是同学院合作开办的比赛，可到后面学院连教室也不肯挪一间出来了，呵……
　　我并非这家教育机构的学员，在此之前也没听过这家公司，只是生病瞒着家里把钱花超支了，想拿奖补贴生活费。
　　可这次真是见识到了资本家的丑恶嘴脸，一家教育培训机构举办的比赛果真不是什么好鸟，挂羊皮卖狗肉，事后看了他们官网，更是连自己公司的介绍都没写好。
　　公司在创业园某驾校办公室里开了一隔间当作办公室，而决赛竟然在另一家考研机构的教室里举办。
　　所谓的一等奖“1000 元 + 学习基金”，颁给了一个直接用英文模板，全是英文连翻译都懒得做的项目。
　　只见讲解人上场谈情怀，PPT 吹的天花乱坠，跟网站内容一点关系没有，全程搞笑尬聊。评委低头玩手机，还去上厕所，一点没认真听，您能评个一等奖出来我也是服气。
　　想来获奖者的安排也考虑了能否收回成本吧，毕竟奖品中有所谓“学习基金”，若让能力强者获奖了，定然不稀罕去他们那里培训了，发的奖金不就打水漂了。况且仅我是个人参赛，钱全归于我一人身上，不像那些几人小组，学习基金分下去，每人只得一点，想报班还得自己另掏钱补，一下就回本了。回想起来，赛前的培训也是胡闹，第一次培训花了两个小时才讲了 Html 的几个标签，第二次直接介绍起了 Bootstrap？？？第三次照着官方文档讲了 Bootstrap 的几个组件，让复制粘贴自己动手试试，内容一切详尽还用得着您教？培训结束时终于露出马脚，宣传起了“价值**元的 Java 试听课”，还说对网页制作有帮助？？？这便是胡闹似的三次培训。
　　怪我不知藏拙。想来如果我在汇报时不谈自己如何如何，而说“在老师的三次培训中受益良多，激发了我学习前端的热情，希望能跟着兴唐的老师们继续学习”，把欲报班学习的欲望展露足了，装作一只不谙世事的绵羊，这奖就到手了吧。
　　至于我所得“优秀奖”的丰厚奖品为一块零几年产的U盘，厂家卖不掉的库存货，他们办公室里有一箱，批发下来一个不知几块钱。
　　年轻气盛，这满腔愤懑不得发泄，在此表露一下不公。
