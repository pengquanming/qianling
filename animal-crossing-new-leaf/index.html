《来吧！动物森友会》3DS 模拟器指北

发表于 2020-04-08 更新于 2020-06-02 分类于 游戏 阅读次数 评论数

Switch 版简直卡爆啦 ><

还是用 3DS 版的体验一下叭~~

模拟器： Canary.7z

游戏文件： dongwuzhisen.cci

汉化补丁： 汉化补丁.zip


使用方法

  1. 解压「Canary.zip」和「汉化补丁.zip」到任意位置

  2. 将解压出来的「user」文件夹移动到「Canary」文件夹内

  3. 在「Canary」文件夹内新建「ROMs」文件夹，将游戏文件「dongwuzhisen.cci」移动进来

  4. 打开「Canary」文件夹中的「canary1550.exe」

  5. 双击「Add New Game Directory」，添加「ROMs」文件夹

  6. 双击列表中的「ACNL- Welcome amiibo」开始游戏，完成教学引导后即可存档

  7. 创建「canary1550.exe」的快捷方式至桌面

　　这个版本的模拟器是贴吧大佬 @娜娜米小时候 制作的本地版！！！缓存文件不会放在 C 盘占空间！！还是加载最快的版本喔！！启动程序是橙子图标 canary1550.exe ~~

　　要把语言和地区设置为日本才能显示中文唷！

　　Mii Maker 是任天堂 3DS 制作的玩家角色工具（系统自带）。玩家可在玩游戏的时候把 Mii 当成自己的分身来操纵，或者是当发送讯息时的图示。很多游戏需要 Mii Maker 制作的玩家角色（Mii）来创建新存档，动森也可以用 Mii 捏的娃娃！

附一

按住 S 键可以奔跑
对话中按下 S 键可以加快说话速度，一句话按一次
使用捕虫网时按住 A 键可以悄悄接近
同时按下 Q 键 和 W 键可以拍照，照片位置在「/Canary/user/sdmc/DCIM/100NIN03」
按下 H 键可以切换道具，按下 G 键收回道具
按下 M 键保存，每次读档会回到自宅门口，所以如果走远或迷路了，保存后重新读档是个方便的办法唷~
拖动同类型水果可以合并成一组。

附二

名字建议设置为英文，否则联机时外国玩家只能看见 ??，一字节一问号
如果存档已经玩了很久了，想要迁移数据到新档，可以重新安装一个全新的模拟器，打开新旧两个模拟器，进入同一房间联机交换物品。

附三

神灯精灵「幽太郎」附身「Amlibo」方法
下载文件： Amlibo.zip
解压后把「Animal Crossing」文件夹放在「Canary」文件夹里
在模拟器菜单中选择「文件-Amlibo-加载」，随便选择一个「.bin」文件就可以啦！
如果不喜欢这个造型的话，再次点击「文件-Amlibo-加载」，换一个试试叭（重启模拟器才会生效喔）~

附四

游戏币：
用于购买幸运饼干的游戏币的正常获得方式是在现实中携带 3DS 设备行走，模拟器则可以修改内存取得，打开「模拟-设置-系统-游戏币」修改个数就可以啦

附五

为什么启动游戏时加载的 Shaders 越来越多了？
理论上来说 Shaders 相当于解析好的缓存文件，游戏中加载过的资源、场景会自动生成为 Shaders，这样在第二次加载时就变快啦！
但是……emmm 在我严谨的计时测试之下，动森不论开与不开，「场景切换时间」都是一样的！所以还是关了叭 !!!∑(ﾟДﾟノ)ノ
关闭方法：取消「模拟-设置-图形-Use Disk Shader Cache」的勾选。

附六

请不要阅读任天堂邮件里的说明书！游戏会宕机！

附七

联机方法

  1. 加入房间：点击模拟器右下角的「未连接，点击此处查找一个房间」，双击进入一个公共房间，首选游戏必须为「ACNL-Welcome amibo」。连接成功后，进入车站，选择选项即可，只能「本地通信」
  2. 创建房间：打开 Citra Community 注册账号，注册成功后复制「login token」，粘贴到「模拟-设置-网络-令牌」，点击「验证」，然后点击「多人游戏-创建房间」就好啦，打开「多人游戏-浏览公共游戏大厅」就可以看到了喔！
  3. [CN] ACNL Citra Pals Server：https://discord.gg/sRB7KAG

附八

手机版 Animal Crossing: Pocket Camp 的变化：

  • 更加功利，与 3DS 版的风格不同
  • 画质更好
  • 界面更友好
  • 无中文版，英语口语词汇很多，用来学英语挺好哒

附九

家具摆放规则
东方红：在房间东侧（进门右手边）2 格宽范围内摆放“红色家具”会带来“道具运”。
西方黄：在房间西侧（进门左手边）2 格宽范围内摆放“黄色家具”会带来“金钱运”。
南方绿：在房间南侧（进门背后侧）2 格宽范围内摆放“绿色家具”会带来“整体运”。
交叠处：西南角（进门背后左侧角落）4 格交叠处摆放“黄绿色家具”、东南角（进门背后右侧角落）4 格交叠处摆放“红绿色家具”也会提升整体气运。
