iOS 14 越狱常见问题

发表于 2021-05-02 分类于 iPhone 阅读次数 评论数

网络异常、相册异常

Safari、知乎、百度等应用均卡在加载页，只有 QQ、微信正常／拍照后照片无法保存至相册、App 无法读取相册。
使用「连个锤子」、「乐网」、「VPN」、「还原网络设置」等方法均无法解决。

解决办法：

  1. 添加壹筒天下源：https://yttxcs.com/
  2. 安装“软重启补丁「系统」”，每次重新越狱后均需再次安装此插件以软重启

装错插件无法再次越狱

  1. 打开「unc0ver」，关闭「Settings-Load Tweaks」，点击 Done 回到首页，点击「Jailbreak」越狱
  2. 越狱成功后，打开「Cydia」删除不兼容插件，重启手机恢复未越狱状态
  3. 重新打开「unc0ver」，开启「Settings-Load Tweaks」，点击 Done 回到首页，再次越狱即可

Cydia 闪退

重启恢复未越狱状态，打开「unc0ver」，开启「Settings-Reinstall Cydia」，点击 Done 回到首页，再次越狱即可

「System files missing!」弹窗

  1. 添加软件源「yilai.yttxcs.com」
  2. 更新「RocketBootstrap」至 1.0.10~beta1 版本

unc0ver 自签

打开 http://dl.ix10.cn/CIJ
按步骤操作即可。

打开「Settings-Advanced」，将「Check Expiry Times」改为「Every 24 Hours」，可减少耗电。

终端命令：
取消后台常驻自动签名：launchctl unload -w /Library/LaunchDaemons/jp.soh.reprovisiond.plist
恢复自动签名：launchctl load -w /Library/LaunchDaemons/jp.soh.reprovisiond.plist

备份黑解

  1. 用「Filza」打开 var/wireless/Library/Preferences/com.apple.commcenter.device_specifi_nobackup.plist
  2. 展开 kPostponementTicket，复制 ActivationTicket 的值，留存备份
  3. 刷机、抹除后需要复原黑解时，越狱恢复该串码，重启手机后生效
