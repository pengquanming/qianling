Hosts

发表于 2017-06-10 更新于 2020-01-06 分类于 网络 阅读次数 评论数

2020/01/06 更新

Hosts 存放域名和其对应的IP地址。

当我们在浏览器在地址栏中输入网址并回车，系统会首先检查本地的 Hosts 文件，看是否已经有相应的域名与 IP 地址的对应关系，如果有就会直接采用，如果没有，那么就得由 DNS 服务器解析域名获取 IP。

防火长城针对某些域名进行 DNS 污染，所以使我们无法解析到一个“正常”的 IP 地址。

所以……

本文介绍几个开源的且更新比较及时的 Hosts。

如何选择一个 Hosts 呢？

  • 更新频率
  • 使用的用户数量

修改Hosts

手动方法

Windows

用文本编辑器（如记事本等）打开文件：C:\Windows\System32\drivers\etc\hosts

并将hosts文件中的内容全部内容复制到该文件中并保存。

注意：如果遇到无法保存，请右键文件Hosts并找到“属性” -> “安全”，然后选择你登陆的用户名，最后点击编辑，勾选“写入”即可。

生效方法：开始 -> 运行 -> 输入cmd -> 在CMD窗口输入ipconfig /flushdns

Mac OS

修改 /private/etc/hosts 文件，并将我们的hosts文件中的内容全部内容复制到该文件中并保存。

也可以使用GAS MASK工具。

Android

修改 /system/etc/hosts 文件，并将hosts文件中的内容全部内容复制到该文件中并保存。

iOS

修改 /etc/hosts 文件，并将hosts文件中的内容全部内容复制到该文件中并保存。

Linux

修改 /etc/hosts 文件，并将hosts文件中的内容全部内容复制到该文件中并保存。
